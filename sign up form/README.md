# Signup Form

A simple, static signup form layout created to practice HTML structure and CSS styling.

## 🚀 Features
* Clean, minimalist design.
* Semantic HTML5 form elements.
* Responsive CSS layout.

## 💻 Technologies
* **HTML5:** Form labels, inputs, and structure.
* **CSS3:** Flexbox for centering, custom styling for form fields.

## 📦 Usage
Simply open the `index.html` file in your preferred web browser to view the form.
